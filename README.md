# Newsletter

## Simple single page application for sharing news articles

## Functionality
* ## Two types of users - admin and regular
    * ### Admin 
        * Creating articles
        * Editing Articles
    * ### Regular user
        * Writing comments 
        * Replying to comments
        * Changing profile picture 
        * Changing password and email
